# 🎵 Song.zreamer.top — 音乐站点

## 简介

**Song.zreamer.top** 是由开发者 **Zreamer** 打造的个人音乐网站，采用简洁的深色主题设计，专注于提供纯粹的音乐浏览与播放体验。

## 核心特点

| 特性 | 说明 |
|------|------|
| 🌙 **深色模式** | 全站采用深色背景，护眼且富有科技感 |
| 🎶 **音乐聚合** | 提供在线音乐资源访问与播放功能 |
| ⚡ **轻量快速** | 基于静态页面构建，加载迅速 |
| 🔓 **开源免费** | GitHub 开源项目，完全免费使用 |
