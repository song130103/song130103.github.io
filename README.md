# 🎮 Song.zreamer.top — 个人站点

## 简介

**Song.zreamer.top** 是由开发者 **Song**（网名）打造的个人专属站点，"Song" 即开发者本人的姓氏。网站专注于提供轻量化的在线小游戏与实用导航服务，采用简洁的深色主题设计。

## 核心特点

| 特性 | 说明 |
|------|------|
| 🌙 **深色模式** | 全站采用深色背景，护眼且富有科技感 |
| 🎮 **小游戏聚合** | 内置多款经典在线小游戏，即点即玩 |
| 🧭 **实用导航** | 整合精选外链资源，快速跳转 |
| ⚡ **轻量快速** | 纯静态页面构建，加载迅速 |
| 🔓 **开源免费** | 有轻微广告、无登录、完全免费开放 |

## 站点内容

- **在线小游戏**：小鸟闯关、谷歌小恐龙复刻版、2048 经典版
- **实用外链**：安卓工具资源入口、网站主站（朋友网站导航）
